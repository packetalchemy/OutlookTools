## ۱. معرفی

**OutlookTools** یک افزونه **Open-Source** و **رایگان** برای Microsoft Outlook است. این ابزار امکاناتی را اضافه می‌کند که اوت‌لوک به صورت پیش‌فرض ندارد.

**مزیت اصلی:** چون کد باز است، می‌توانید قبل از نصب کد را بخوانید و مطمئن شوید هیچ داده‌ای آپلود نمی‌شود.

### ۱۳ قابلیت:

| # | قابلیت | توضیح |
|---|--------|-------|
| ۱ | Reply with Attachments | پاسخ با حفظ پیوست‌ها |
| ۲ | Reply All with Attachments | پاسخ به همه با حفظ پیوست‌ها |
| ۳ | Forward without Attachments | فوروارد بدون پیوست |
| ۴ | Advanced Search | جست‌وجوی پیشرفته |
| ۵ | Smart Archive | بایگانی هوشمند فصلی |
| ۶ | Reminder Cleanup | پاک‌سازی یادآورها |
| ۷ | Email Templates | قالب‌های ایمیل |
| ۸ | Bulk Actions | عملیات گروهی |
| ۹ | Email Statistics | آمار ایمیل |
| ۱۰ | **Follow-up Tracker** ⭐ | پیگیری پاسخ‌ها |
| ۱۱ | **Email Snooze** ⭐ | مخفی‌سازی موقت |
| ۱۲ | **Daily Digest** ⭐ | خلاصه روزانه |
| ۱۳ | **Quick Notes** ⭐ | یادداشت سریع |

## ۳. نصب و راه‌اندازی

### نصب از سورس:
1. Visual Studio 2019/2022 را باز کنید
2. `OutlookTools.sln` را باز کنید
3. `Ctrl+Shift+B` برای Build
4. فایل VSTO در `bin/Debug/` اجرا کنید
5. اوت‌لوک را باز کنید

### آپدیت:
نسخه جدید VSTO را روی نسخه قبلی اجرا کنید. نیازی به حذف نیست.

### حذف:
Settings → Apps → OutlookTools → Uninstall

## ۵. پاسخ با پیوست

**مراحل:**
1. ایمیل را انتخاب کنید
2. روی **Reply with Attachments** کلیک کنید
3. پاسخ با پیوست‌ها باز می‌شود

**تفاوت Reply vs Reply All:**
- **Reply with Attachments:** فقط به فرستنده + پیوست‌ها
- **Reply All with Attachments:** به همه + پیوست‌ها

> تصاویر امضای درون‌خطی حفظ می‌شوند.

---

## ۶. فوروارد بدون پیوست

**مراحل:**
1. ایمیل را انتخاب کنید
2. روی **Forward without Attachments** کلیک کنید
3. فوروارد بدون فایل‌ها باز می‌شود

> فقط فایل‌های واقعی حذف می‌شوند، تصاویر امضا باقی می‌مانند.

## ۷. جست‌وجوی پیشرفته

**فیلترها:**
| فیلتر | توضیح | مثال |
|-------|-------|------|
| From | فرستنده | ali@company.com |
| To | گیرنده | reza |
| Subject | موضوع | گزارش |
| Body | متن | قرارداد |
| Date | بازه زمانی | از ۱۴۰۴/۰۱/۰۱ |
| Has attachment | پیوست دارد | ✅ |

**مراحل:**
1. Advanced Search را باز کنید
2. فیلترها را پر کنید
3. Search را بزنید
4. نتایج را مرور کنید
5. دوبار کلیک برای باز کردن

> حداکثر ۵۰۰ نتیجه نمایش داده می‌شود.

## ۱۰. قالب‌های ایمیل

**Placeholders:**
| Placeholder | جایگزین |
|-------------|---------|
| `{DATE}` | تاریخ امروز |
| `{TIME}` | ساعت فعلی |
| `{DATETIME}` | تاریخ و ساعت |
| `{YEAR}` | سال جاری |
| `{MONTH}` | نام ماه |

**مثال:** `Hello {DATE}` → `Hello 2026-07-16`

**ذخیره:** `%LOCALAPPDATA%\OutlookTools\Templates\`

## ۱۳. پیگیری پاسخ‌ها ⭐

> **جدید در v1.2.0**

ایمیل‌های ارسال شده را پیگیری می‌کند و اگر جوابی نیامد یادآوری می‌دهد.

**ویژگی‌ها:**
- ✅ خودکار همه ایمیل‌های ارسال شده را ردیابی می‌کند
- ✅ هر ۳۰ دقیقه بررسی پاسخ‌ها
- ✅ داشبورد با لیست پیگیری‌های باز
- ✅ هشدار برای پیگیری‌های دیرشده
- ✅ Snooze / Resolve / Remove
- ✅ خروجی CSV

**چطور کار می‌کند:**
1. هر ایمیلی که ارسال می‌کنید خودکار ردیابی می‌شود
2. هر ۳۰ دقیقه Inbox بررسی می‌شود
3. اگر پاسخی با همان Subject یا ConversationID پیدا شد → Resolve
4. اگر بعد از X روز پاسخی نیامد → Overdue

3. در زمان مشخص شده به Inbox برمی‌گردد
