# OutlookTools Installer

## نصب:
1. فایل‌ها رو در یک پوشه قرار بدید
2. روی `Install.bat` راست کلیک کنید
3. **Run as Administrator** رو بزنید
4. صبر کنید تا نصب تمام شه
5. اوت‌لوک رو باز کنید

## حذف:
1. روی `Uninstall.bat` راست کلیک کنید
2. **Run as Administrator** رو بزنید

## نکات:
- نصب‌کننده نیاز به **Admin Rights** داره (فقط برای ثبت DLL)
- فایل‌ها در `%LOCALAPPDATA%\OutlookTools` ذخیره می‌شن
- تنظیمات در `settings.json` محلی ذخیره می‌شن
- هیچ چیزی در سیستم‌عامل تغییر نمی‌کنه (فقط Registry برای COM)
